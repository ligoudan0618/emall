    1、express
    2、mongoose
    3、cors 跨域包
    4、multer 图片上传
    5、inflection 将小写单词转成大写，将负数改为单数
    6、require-all 获取所有模型
    7、bcrypt 对密码加密
    8、http-assert 响应错误信息 如果 express 版本过低 无法捕获  解决：安装高版本 express
